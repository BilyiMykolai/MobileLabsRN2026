## Опис проєкту

Мобільний застосунок «Файловий менеджер» для роботи з локальною файловою системою пристрою з використанням бібліотеки `expo-file-system`.

### Реалізований функціонал (згідно з завданням):

1. **Навігація**
   - Відображення поточного шляху (breadcrumb)
   - Перехід у вкладені папки
   - Перехід назад (кнопка «Назад»)
   - Відображення вмісту директорії через `FlatList`

2. **Створення**
   - Створення нової папки (модальне вікно)
   - Створення `.txt` файлу з іменем та початковим вмістом

3. **Зчитування**
   - Відкриття `.txt` файлів для перегляду вмісту

4. **Редагування**
   - Редагування текстових файлів
   - Збереження змін

5. **Видалення**
   - Видалення файлів та папок
   - Підтвердження перед видаленням (Alert)

6. **Детальна інформація**
   - Назва файлу
   - Тип (за розширенням)
   - Розмір
   - Дата останньої модифікації

7. **Статистика пам’яті**
   - Загальний обсяг
   - Вільний простір
   - Зайнятий простір (з прогрес-баром)
